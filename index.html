Proyectos

Mis conocimientos plasmados en proyectos que resuelven necesidades reales

Data Lovers

Proyecto, donde transformé la data JSON en información, construyendo así una página web de Pokemón donde el usuario podrá visualizar la data, filtrarla, ordenarla y hacer cálculos. Proyecto donde tuve en consideración las necesidades de mi público objetivo para que sea de fácil manipulación y agradable a la vista.
Herramientas utilizadas: Arrays, Objects, DOM, XHR/fetch, Figma, Visual Design, Jest.

Red Social

Proyecto, donde elaboré una página web con más de una vista, haciendo uso del patrón de diseño MVC. Proyecto que me ayudó a separar los datos, la lógica y la interfaz del usuario, teniendo como plataforma principal Firebase.
Herramientas utilizadas: Mobile first, Responsive, URL, HTTP, Múltiples vistas, Routing, Escritura de datos, Firebase.
